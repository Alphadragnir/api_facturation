# 📊 Projet de Gestion de Facturation - Django

## 🚀 Description
Application web de gestion de facturation développée avec **Django** et **Python** permettant de gérer les produits, les factures et les utilisateurs.

---

# ⚙️ Installation du projet

## 1. Décompresser le projet
Décompressez le fichier :

```
gestion_facturation_django_final_complete.zip
```

Puis entrez dans le dossier :

```bash
cd gestion_facturation_django
```

---

## 2. Créer l’environnement virtuel

```bash
python -m venv venv
```

---

## 3. Activer l’environnement virtuel

### 🪟 Windows PowerShell
```bash
.\venv\Scripts\Activate.ps1
```

### 🪟 Invite de commandes Windows (CMD)
```bash
venv\Scripts\activate.bat
```

### 🍎 macOS / Linux
```bash
source venv/bin/activate
```

---

## 4. Installer les dépendances

```bash
pip install Django reportlab
```

---

## 5. Appliquer les migrations

```bash
python manage.py migrate
```

---

## 6. Démarrer le serveur

```bash
python manage.py runserver
```

---

## 🌐 Accès à l’application

http://127.0.0.1:8000
